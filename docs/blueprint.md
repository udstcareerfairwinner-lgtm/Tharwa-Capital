# **App Name**: Barakah Invest

## Core Features:

- Bilingual Support: Full support for both Arabic and English languages, including RTL layout for Arabic.
- Sharia Compliance Filter: Allow users to filter investments based on Sharia compliance certifications. LLM tool assists in reviewing compliance reports, identifying key considerations and ensuring alignment with Sharia principles.
- Investment Marketplace: Browse and filter fractional investment opportunities in real estate, small businesses, and renewable energy.
- Community Forum: A dedicated community section where investors can discuss opportunities and share insights. Integrated translation for Arabic and English.
- Onboarding Process: Step-by-step onboarding to educate new users about fractional investing and Sharia compliance.
- User Dashboard: Personalized dashboard with portfolio overview, investment returns, and active projects.
- Sharia Advisory Panel Profiles: Introduce the Sharia advisory board members, highlighting their expertise and qualifications.

## Style Guidelines:

- Primary color: Emerald green (#34D399) to represent growth and Islamic values.
- Background color: Light green (#F7FEE7), a very subtly tinted off-white to visually set off app content.
- Accent color: Gold (#FBBF24) for highlights and calls to action, symbolizing prosperity and tradition.
- Body font: 'Alegreya', a serif for readability and a literary feel. Headline Font: 'Belleza', a sans-serif chosen for personality and fashion.
- Use Islamic-inspired geometric motifs and clean, modern icons.
- Clean, minimalist fintech UI with a focus on clarity and ease of use. RTL layout support for Arabic content.
- Subtle animations to provide feedback on interactions and enhance user experience.
